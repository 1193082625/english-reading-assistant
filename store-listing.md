# Chrome Web Store Listing

## Short Description (132 chars max)

English reading assistant: translate, interpret long sentences, and generate summaries with AI.

## Detailed Description

English Reading Assistant helps you read English documents more efficiently. Select any text on a webpage to translate, analyze grammar, or generate summaries.

### Features

- **Translate** — Select text and translate between English and Chinese (bidirectional). Uses MyMemory free API by default, falls back to AI translation automatically.
- **Interpret** — Long sentence grammar analysis with structure breakdown, verb tense analysis, and clause explanation. Auto-detects code content and switches to technical explanation mode.
- **Summarize** — Generate structured summaries for selected text or entire pages (key points, conclusions, API lists, etc.).
- **Custom Terminology** — Define your own term translation mappings for consistent technical terminology.
- **Right-click Translation** — Right-click any paragraph to quickly translate.
- **Dark Mode Support** — Automatic dark mode adaptation.

### How to Use

1. Open any English webpage
2. Select the text you want to process
3. Click the floating "译" button at the bottom right
4. Switch between Translate / Interpret / Summary tabs
5. Right-click paragraphs for quick translation

### AI Service Configuration

Click the extension icon to configure your preferred AI provider: DeepSeek, MiniMax, Ollama (local), or any OpenAI-compatible API.

- Default: DeepSeek (requires API key)
- Translation falls back to free MyMemory API when no key is configured
- Local Ollama requires no API key

### Privacy

This extension only sends selected text to third-party services when you actively trigger an action. All settings are stored locally in your browser. No personal data is collected, and no browsing history is tracked.
