# 隐私政策

**英文阅读助手**（English Reading Assistant）尊重并保护用户隐私。本政策说明了扩展程序如何收集、使用和存储用户数据。

## 数据收集与使用

### 翻译与 AI 服务

当您使用以下功能时，选中的文本会被发送到第三方服务进行处理：

- **翻译功能** — 文本发送至 MyMemory API (`api.mymemory.translated.net`)
- **AI 解读/摘要** — 文本发送至您配置的 AI 提供商（DeepSeek / MiniMax / Ollama 等）
- **整页分析** — 页面正文内容发送至您配置的 AI 提供商

这些服务仅在您主动触发操作时才会调用（选中文本 → 点击翻译/解读/摘要按钮）。

### 设置存储

以下数据存储在您的浏览器本地（`chrome.storage.local`），**不会**上传到任何服务器：

- API 地址、API Key、模型名称
- 术语翻译映射配置
- 默认功能 Tab 设置

### 不收集的信息

本扩展**不会**：

- 收集个人身份信息（姓名、邮箱、地址等）
- 跟踪浏览历史
- 收集统计信息或使用数据
- 在后台发送网络请求（仅在您主动操作时调用 API）
- 使用 Cookie 或类似追踪技术

## 数据保留

- 设置数据仅保存在本地浏览器存储中
- 翻译和 AI 结果会在当前页面会话期间缓存（页面刷新后清除）
- 卸载扩展后所有本地存储的数据将被清除

## 第三方服务

本扩展使用以下第三方服务，分别受其各自隐私政策约束：

| 服务 | 用途 | 隐私政策 |
|------|------|---------|
| MyMemory API | 免费翻译 | https://mymemory.translated.net/doc/privacy |
| DeepSeek API | AI 翻译/解读/摘要 | https://www.deepseek.com/privacy |
| MiniMax API | AI 翻译/解读/摘要 | https://cdn.minimaxi.com/minimax/agreement/privacy.html |

配置为 Ollama（本地）时，数据不会离开您的设备。

## 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 读取当前页面选中的文本内容 |
| `storage` | 在本地保存您的设置和术语库 |

## 更新

本隐私政策可能不时更新。如有重大变更，将在扩展更新时通知用户。
